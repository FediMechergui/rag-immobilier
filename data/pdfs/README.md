# Place your PDF files here
# Supported languages: French, English, Arabic

This folder is where you should place your PDF documents for the RAG pipeline to process.

## How to use:

1. Copy your PDF files into this folder
2. Run the ingestion command:
   ```bash
   python main.py ingest
   ```
3. Or use the web interface to upload files directly

## Supported formats:
- PDF files (.pdf)
- Text-based PDFs work best
- Scanned PDFs may require OCR preprocessing
